Существует несколько типов **cashback** бонусов, которые могут быть предложены игрокам:

- **Порог потерь для cashback** — чтобы получить cashback, игрок должен достичь определенной суммы потерь, например, 1000 рублей за неделю.

1. Выполнить условия, установленные казино для получения возврата (например, потерять определенную сумму за неделю).
2. Ожидать, пока бонус будет автоматически начислен на ваш игровой счет (иногда его нужно активировать вручную).
3. Использовать полученный cashback для новых ставок или вывести деньги, если бонус не имеет условий отыгрыша.
